# Alura-Latam-DataScience-Challenge-1

# 📊 Análisis de Eficiencia Operativa: Alura Store 🛒

Este proyecto forma parte de un desafío de **Data Science** orientado a la toma de decisiones estratégicas. El objetivo es asesorar al Sr. Juan, dueño de la cadena **Alura Store**, para identificar cuál de sus 4 tiendas es la menos eficiente y debería ser vendida para financiar un nuevo emprendimiento.

## 🎯 Objetivo del Proyecto
Analizar el rendimiento integral de 4 tiendas mediante el procesamiento de datos en Python, evaluando métricas financieras, logísticas y de satisfacción del cliente para emitir una recomendación de venta basada en evidencia.

---

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python
* **Librerías:** * `Pandas`: Para la carga, limpieza y manipulación de datos (DataFrames).
    * `Matplotlib`: Para la creación de visualizaciones (barras, dispersión, pasteles).
* **Entorno:** Google Colab / Jupyter Notebook.

---

## 📈 Metodología de Análisis
El análisis se dividió en 5 pilares fundamentales:

1.  **Análisis de Facturación:** Comparativa de ingresos totales para medir el peso económico de cada tienda.
2.  **Mix de Categorías:** Identificación de las categorías de productos con mayor y menor tracción comercial.
3.  **Satisfacción del Cliente:** Evaluación de las calificaciones promedio (estrellas) para detectar riesgos reputacionales.
4.  **Desempeño de Productos:** Listado de los productos "Best Sellers" y aquellos con nula rotación de inventario.
5.  **Eficiencia Logística:** Cálculo del costo de envío promedio para identificar ineficiencias en la cadena de suministro.
